## 実行したコード

```python
import gym
import numpy as np
import matplotlib.pyplot as plt
import matplotlib.animation as animation

env = gym.make('SpaceInvaders-v4')
print('Observation Space: ', env.observation_space)
print('Action Space: ', env.action_space)
print('Action meanings: ', env.unwrapped.get_action_meanings())

# 初期状態の確認
observation = env.reset()

frames = []
histories = []


observation = env.reset()

fig = plt.figure()



for step in range(1000):
    frame = plt.imshow(observation)
    frames.append([frame])

    action = env.action_space.sample()
    observation_next, reward, done, info = env.step(action)
    observation = observation_next
    histories.append([observation_next, reward, done, info] )

    if done:
        print('step:', step)
        break

ani = animation.ArtistAnimation(fig, frames, interval=50)
plt.show()
```

### 3.done
ゲーム終了時のみTrue、それ以外はFalseになる。

例えば、
```python
where_done = [i for i, x in enumerate(don) if x == True]
where_done[0] #donがTrueになるのは1回しかないので、このリストの長さは必ず1になるはず。
```

```python
out[]:
758
```
みたいな感じでゲームが終了したstepを取り出せる。

### 4.info

```python
inf[1]
```

```python
Out[]:
 {'ale.lives': 3}
```

みたいな感じで、各々は辞書型で、

key をale.livesとして、step後の残機数のvalueが含まれている。

